# 资源结构

生成的目录树如下

``` text
arknights                   // 资源总目录   
├─ assets                   // 图片/文字等资源
│  ├─ product               // 合成的图片成品
│  │  ├─ mastery            // 技能专精
│  │  ├─ promotion          // 精英化
│  │  └─ skill              // 技能
│  └─ resource              // 原始资源
│     ├─ background                   
│     ├─ others                       
│     ├─ professions                  
│     └─ subProfessions  
├─ db                       // 游戏数据库                
└─ record.json              // 记录文件
```
